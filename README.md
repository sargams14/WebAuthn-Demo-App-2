
# WebAuthn Demo App

A Spring Boot application to explore how FIDO2 works to enable login using Passkeys with the help of the WebAuthn4j library.

## Author

- [@sargams14](https://www.github.com/sargams14)


## Functionalities

- Register a new user's credential
- Authenticate the user with the help of the passkey
- The app contains simple UI made with Vanilla JS and HTML
